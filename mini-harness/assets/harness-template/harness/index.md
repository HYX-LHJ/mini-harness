# Mini Harness

本目录存放面向编码 Agent 的、紧凑的仓库本地状态。

| 入口 | 用途 |
|------|------|
| [PROGRESS.md](PROGRESS.md) | 当前事实、验证状态与阻塞项 |
| [todo.md](todo.md) | 面向验收的进行中任务 |
| [DECISIONS.md](DECISIONS.md) | 按主题记录的重大决策（迭代取舍） |
| [skills/](skills/) | 已迁移到仓库的内置 Agent Skills |
| [rules/](rules/) | mini-harness 自带的通用编码规范 |
| [scripts/](scripts/) | 仓库本地的 harness 管理脚本 |
| [plans/](plans/) | 待批准或已记录的重要设计 |
| [acceptance/](acceptance/) | 验收报告（对照 todo AC） |
| [code_review/](code_review/) | 评审报告与未关闭发现 |
| [code_simplifier/](code_simplifier/) | 可选的简化记录 |
| [backlog/](backlog/) | 已归档的任务与决策历史 |
| [profile/](profile/) | 项目画像与进化记录（项目自有，插件不覆盖） |

仓库专属文档与自动化可扩展本目录；`profile/` 由项目维护并随协作进化。
