# Mini Harness

本目录是面向编码 Agent 的**紧凑协作状态** — 文件即记忆，不赌聊天历史。

| 入口 | 用途 |
|------|------|
| [PROGRESS.md](PROGRESS.md) | 📍 当前事实、验证、阻塞 |
| [todo.md](todo.md) | ✅ 进行中任务 + AC |
| [DECISIONS.md](DECISIONS.md) | 🏛️ 按主题的重大决策 |
| [profile/](profile/) | 🎯 项目画像（每回合必读） |
| [skills/](skills/) | 内置 Agent Skills |
| [scripts/](scripts/) | install / update / doctor |
| [plans/](plans/) | 待批准的设计 |
| [acceptance/](acceptance/) | 验收报告 |
| [code_review/](code_review/) | 审查报告 |
| [code_simplifier/](code_simplifier/) | 精炼记录 |
| [backlog/](backlog/) | 已归档任务 |

**每回合入口：** [skills/using-harness/SKILL.md](skills/using-harness/SKILL.md)

`profile/` 由项目维护，`update` 不覆盖。
