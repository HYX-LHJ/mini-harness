# mini-harness-zh（中文版 Skill）

[mini-harness](../README.zh-CN.md) 的中文 Agent Skill 包。

| 语言 | Skill 包 |
|------|----------|
| 中文（本包） | `mini-harness-zh/` |
| English | [`mini-harness-en/`](../mini-harness-en/) |

## 安装

```bash
npx skills add HYX-LHJ/mini-harness --skill mini-harness-zh -g -y
```

或复制 `mini-harness-zh/` 到 `~/.cursor/skills/`、`~/.claude/skills/` 等。

## 使用

> 用 mini-harness-zh 在当前仓库创建 harness

初始化产物（`AGENTS.md`、`harness/`）默认为**中文**。
